# Profit Clients Web App V12 — Month Dashboard + Full Edit Controls

نسخة مبنية على الملفات التي أرسلتها أنت، بدون إزالة أي جزء من الموقع.

## الجديد في V12
- تثبيت عمود **إجراءات** في الجداول العريضة، خصوصًا الفواتير، بحيث زر التعديل يفضل ظاهر حتى لو الجدول طويل.
- تقوية أزرار التعديل والحذف لكل الأقسام الأساسية:
  - العملاء
  - الباقات
  - الفيديوهات
  - الفواتير
  - الاستلامات السريعة
  - المدفوعات اليدوية
  - المصروفات
  - التحويلات
- إضافة إجراءات أقوى للفواتير:
  - تعديل الفاتورة
  - تسجيل استلام عليها مباشرة
  - طباعة الفاتورة
  - حذف الفاتورة
- إضافة زر تعديل من صفحة طباعة الفاتورة نفسها.
- الحفاظ على نظام الشهور كما هو:
  - فتح شهر جديد
  - إقفال شهر
  - إعادة فتح شهر قديم
  - الرجوع لأي شهر والعمل عليه
  - تعديل الرصيد الافتتاحي للشهر المفتوح
- لا تحتاج أي SQL جديد في Supabase إذا كانت المزامنة تعمل بالفعل.
